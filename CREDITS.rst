=======
Credits
=======

This file records the known human history of ``acdcontrol`` across its
SourceForge origin, GitHub fork lineage, and current maintainer era.

Addresses are included only when they are explicitly recorded in a public
project artifact such as source headers, copyright files, or public commit
metadata. No email addresses are inferred from usernames.

Original developer
==================

* Pavel Gurevich <faxik@users.sourceforge.net>

  Original developer of the SourceForge project
  ``Apple Cinema Display backlight control``.
